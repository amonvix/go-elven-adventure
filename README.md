# Go - The Elven Adventure 🧝⚔️

A sacred language forged by geniuses, mastered through the wisdom of the elves.  

## How it works

The goal is to create an accessible Go learning resource for beginners using an RPG world as the learning environment.

### World Structure

 - Chapters = map regions  
 - Lessons = quests  
 - Short exercises = side quests  
 - Important concepts = relics  
 - Final challenge of the stage = boss fight  
 - Larger project = dungeon / raid

 ## The first area will cover the following locations in Eldora:

 ### 1. The Village of Awakening

 Here you learn about:

 - package main  
 - func main()  
 - fmt.Println  
 - variáveis  
 - declaração curta

## Stage Boss: 

"Create a little terminal with your hero BIO"

### 2. Forest of Decisions

Here you learn about:

 - if  
 - else  
 - switch  
 - operators  
 - comparisons  

## Stage Boss:

"Decide the Hero's fate based on choices and status."

### 3. Cave of Collections

Here you learn about:

 - arrays  
 - slices  
 - maps  

 ### 4. Temple of Forms

Here you learn about:

 - structs  
 - methods  
 - receivers  

 ## Stage Boss:

"Create world entities with attributes and behaviors."
